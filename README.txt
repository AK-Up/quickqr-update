QuickQR - QR Code Generator
===========================

QuickQR is a minimalist and modern QR code generator for desktop.

Features:
- Real-time preview of QR codes
- Custom color support (pattern & background)
- Export to PNG
- Dark theme aesthetic
- Default save path based on OS
- Installer with Start Menu and Desktop shortcut

Usage:
1. Launch the application from your Start Menu or Desktop.
2. Enter the data, file name, and color preferences.
3. Click 'Generate & Save QR' to save the file.
